## Tech Stack (from codebase)

### Mobile App (`measureme`)
- **Framework:** Expo (React Native) v51
- **Language:** TypeScript
- **Auth:** Firebase Auth + Google Sign-In (@react-native-firebase/app, @react-native-firebase/auth, @react-native-google-signin)
- **Navigation:** React Navigation (stack navigator)
- **Storage:** AsyncStorage
- **Date Picker:** @react-native-community/datetimepicker
- **Picker:** @react-native-picker/picker
- **Platform:** iOS + Android
- **Build:** EAS (Expo Application Services)

### Marketing Website (`measureme-website`)
- **Framework:** Next.js (Pages Router)
- **UI:** Tailwind CSS
- **Build:** Yarn

---

## Current State

Mobile app in development with Firebase auth integration. Marketing website built with basic Next.js template.
