# BDC System — FX Rate & Settlement Platform for Bureau De Change

**Status:** FRD Completed, Awaiting Build
**Repo:** None (no codebase yet)

---

## What It Is

An MVP platform for Bureau De Change (BDC) operators to manage FX rates, customer transactions, and back-office settlement workflows. Digitizes the traditionally manual BDC workflow.

---

## User Roles

1. **Customer** — Views daily rates (no login required), views domiciliary accounts, uploads payment receipts, tracks submission status
2. **BDC Owner** — Sets live USD/GBP rates, manages dom accounts, oversees transaction workflow and staff, views analytics dashboard
3. **Back-Office Staff** — Processes customer receipts, manages transaction workflow, communicates status updates to customers

---

## Core Features

- Public FX rate display — customers view current rates without login
- Receipt upload — customers submit transfer receipts (image/PDF) with amount, bank, phone
- Domiciliary account listing — shows BDC's dom accounts
- Structured transaction pipeline — receipt upload → processing → settlement
- Staff assignment and management — every transaction has a responsible handler
- Status notifications — customers receive updates at each stage
- Analytics and summaries — owner dashboard for monitoring operations
