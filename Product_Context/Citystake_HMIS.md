# Citystake HMIS - Hotel/Apartment Management Information System

## Overview
A comprehensive hotel and apartment management system designed to digitize the entire guest lifecycle — from booking through check-out — with smart lock integration, real-time dashboards, and WhatsApp automation.

## Client
**Citystake Apartments** — Abuja-based serviced apartments

## Key Features
- **Digital Guest Registration** — Tablet-based or online forms replacing manual paper logbooks
- **Smart Lock Access Control** — Temporary access codes generated at check-in, sent via WhatsApp (Tuya/SmartLife integration)
- **Real-time Room Status Dashboard** — Live updates: available, occupied, cleaning
- **Unified Booking & Payments** — Automatic invoice generation with integrated Paystack payment tracking
- **BI & Analytics Layer** — Real-time dashboards for occupancy rates and revenue tracking
- **Multi-Device Access** — Role-based terminals for receptionists, managers, and kitchen staff
- **WhatsApp Business API Integration** — Instant notifications, check-in codes, receipts sent automatically
- **Built-in Accounting** — Financial tracking integrated into the management system

## Tech Stack
- **Spacer** — Core booking/workspace management platform
- **Tuya/SmartLife** — Smart lock automation
- **WhatsApp Business API** — Guest communication
- **Paystack** — Payment processing
- **Google Workspace** — Internal operations
- **Airtable** — Optional dashboard/database layer
- **Zapier** — Workflow automation

## Implementation Phases
1. Smart locks setup + digital access code generation
2. Digital registration form (tablet/online)
3. Front desk and management dashboards
4. BI/analytics layer

## Value Proposition
Eliminates paper registration, provides secure online guest records, automates room access with expiring codes, reduces delays through instant notifications, and gives real-time management visibility on occupancy and revenue.

## Competitive Advantage
Even tech-savvy hotels in Abuja lack seamless WhatsApp integration, integrated accounting, smart lock systems, and real-time BI dashboards — this fills that gap.

## Status
Implementation planning — built on Spacer platform.
