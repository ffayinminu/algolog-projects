# CYFT - Facility Health Inspection System

## Overview
A digital platform for conducting comprehensive facility health assessments of hospitals and clinics, generating professional inspection reports. Built for CYFT Consulting Limited.

## System Components
1. **Mobile Inspection App** — On-site data collection by inspectors
2. **Admin Dashboard** — Report generation and management
3. **Client Portal** — Clients view reports and track progress
4. **Report Generator** — Automated PDF/Word report creation

## Scoring System
| Score Range | Status | Color |
|-------------|--------|-------|
| 90-100 | Excellent | Green |
| 70-89 | Good | Yellow |
| 50-69 | Fair | Orange |
| 0-49 | Critical | Red |

## Priority Levels
- **HIGH** — Urgent attention
- **MEDIUM** — Standard priority
- **LOW** — Routine items

## Enhanced Report Features
- Category scores with color-coded badges
- Issues found count per category
- Repair cost estimates (NGN)
- Priority classification
- Last service date tracking
- Total repair/maintenance budget calculation
- Key insights section highlighting immediate attention areas
