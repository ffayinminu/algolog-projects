# BDC System - FX Rate & Remote Settlement Platform

## Overview
An MVP platform for Bureau De Change (BDC) operators to manage FX rates, customer transactions, and back-office settlement workflows.

## User Roles
1. **Customer (End User)** — Views daily rate, views domiciliary accounts, uploads payment receipts, tracks submission status
2. **BDC Owner** — Sets live rates, manages dom accounts, oversees transaction workflow and staff, views analytics
3. **Back-Office Staff** — Processes customer receipts, manages transaction workflow, communicates with customers via status updates

## Core Features
- **Public FX Rate Display** — Customers view current USD/GBP rates without login
- **Receipt Upload** — Customers upload transfer receipts (image/PDF) with amount, bank, phone number
- **Domiciliary Account Listing** — Shows BDC's dom accounts for customer reference
- **Transaction Workflow** — Structured pipeline from receipt upload to settlement
- **Staff Assignment & Management** — Back-office staff process and escalate transactions
- **Status Notifications** — Customers receive updates on transaction progress
- **Analytics & Summaries** — Dashboard for BDC owners to monitor operations

## Value Proposition
Digitizes the traditionally manual BDC workflow, providing transparency, auditability, and operational efficiency for FX transactions.
