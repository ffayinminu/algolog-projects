# BuyMeABeverage - Beverage Gifting/Ordering Platform

## Overview
BuyMeABeverage is a consumer-facing product powered by the SkillPay backend. It operates at **buymeabeverage.com**.

## Backend
Runs on the shared **SkillPay backend** which handles payment processing, user accounts, transaction tracking, and custom link generation.

## Design Workflow
Follows Algolog's standard design pipeline:
1. **Stage 1** — Requirements analysis via ChatGPT
2. **Stage 2** — ShadCN UI design system customization
3. **Stage 3** — Figma Make generation + manual review
4. **Stage 4** — Claude Code conversion to Next.js
5. **Stage 5** — Standard stack integration

SOP Path: B1 -> B2 -> B3

## Status
Built — served by SkillPay backend at buymeabeverage.com.
